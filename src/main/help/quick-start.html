Editing Quick Start

You edit in whatever text editor you want to use. File / New and File / Open assocaite a script file with a window; from then on, when you save the file, the file is re-loaded and executed.

If you have an editor command defined in the Preferences dialog, the application will use it when you associate the file with the window. This is not required. There is a default editor command defined when you first use the programâ€”you can keep it, change it, or remove it.

Script Quick Start

Begin with Help / Sample Scripts... There are a variety of sample scripts you can study, from simple to complex.

A script consists of a series of statements. One type of statement is a command. Commands are things that actually draw something (or control how things are drawn), so focus on them first.

You will need to learn something about objects, but there are three objects you can start with: coordinates, frames, and observers.

Stylesheet Quick Start

Commands control what is drawn; stylesheets control how it is drawn (its appearance).

The stylesheet system is patterned after CSS (Cascading Style Sheets) used by web browsers and has a lot of features (and complexity). There are benefits to using it, but here's a way to get going quickly:

Every command has a style property. The value of this property is a string that contains a series of style properties, separated by semicolons (the final semicolon can be omitted).

The most common style property is color. For instance, write axes style: "color: red"; to draw axes in red. There are many named colors; the most common ("red", "blue", "green", etc.) are supported. If these don't suit you, you can write a color using the same notation as supported by CSS v1 or v2.

For lines (or things with lines, such as axes, or paths), use line-thickness and line-style. The former takes a number (the pixel thickness of the line) and the latter supports styles solid, dashed, and dotted. You can also add arrowheads in some cases with arrow (values are none, both, start, and end).

For events, you might want to set the event-diameter (in pixels) or the event-shape (circle, square, diamond, or star).

For text, a box is drawn around the characters and the box is widened by the text-padding (a value in pixels). This box is then divided into 9 key points. You can select one of these points with a two-letter code: vertically T for top, M for middle, and B for bottom; horizontally L for left, C for center, and R for right. You specify this point with the text-anchor style.
