Shortcuts

Diagram Shortcuts

  • Ctrl + +: Zooms in.
  • Ctrl + -: Zooms out.
  • Ctrl + 0: Resets both zoom and pan to the initial setting (as specified by the display command).
  • Left mouse drag: Pans the diagram.
  • Scroll wheel: Zooms in/out.

Animation Shortcuts

These shortcuts only work when an animated diagram is being viewed:

  • Space: Toggle the animation state between stopped/running.
  • Escape: Stops the animation if it is running.
  • Left arrow: Moves one frame backward when the animation is stopped.
  • Right arrow: Moves one frame forward when the animation is stopped.
  • Up arrow: Speeds up the animation (only works when the animation is running).
  • Down arrow: Slows down the animation (only works when the animation is running).
  • Left brace: Goes to the first frame and stops.
  • Right brace: Goes to the last frame and stops.

Slider Shortcuts

  • Double-click: Resets slider to its initial value.
