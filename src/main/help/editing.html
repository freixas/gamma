Editing a Script

You edit a script using any text editor. Each time the script file—or any dependent file (such as a stylesheet or include file)—is saved, it will be reloaded and executed.

File / New and File / Open simply associate file with a window. The application monitors the associated file for changes.

If you entered an editor command in the Preferences dialog, the application will run the command when a file is first associated with the window. If the editor command is empty, the attempt won't be made.

It is not necessary to have an editor command. The default setting is platform-dependent:

  • Windows: notepad "$F$"
  • Mac: open -t "$F$"
  • Linux: open "$F$"
