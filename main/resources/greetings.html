There are two things you should know about Gamma:

  • You don't create diagrams with the user interface—you write scripts.
  • Gamma doesn't have an editor—you edit the scripts in your favorite text editor.

Scripts

I know. You wanted to try out the program—you didn't want to have to learn a new programming language!

But you're smart or you wouldn't be here. Go to the Help menu and check out the sample scripts. You can probably figure out how things work just by looking at some examples. Then check out the help content which includes some quick start tips.

There is a detailed manual if you need it.

Editing

Before you get too far, go to File / Preferences and look at the editor command. It should be set to a operatng-system-specific editor. Even though Gamma has no editor, when you select a script file, it executes the editor command (replacing "$F$" with the name of the file). You don't have to use the editor command; you can delete it and just open the editor yourself.

Once you point Gamma to a text file, it will update the diagram automatically each time you save it. So, try something, save the file, and see what happens.
