File Menu

New...

Creates a file, associates it with the window, load the empty file, and executes the empty script. From then on, each time the file (or any dependent file) is saved, the file will be re-loaded and executed.

Open...

Opens an existing file, associates it with the window, loads the script, and executes it. From then on, each time the file (or any dependent file) is saved, the file will be re-loaded and executed.

Export Diagram...

Saves the image as a JPG, PNG, or TIFF.

Close Window

Closes the window.

Preferences...

Displays the Preferences dialog:

  • Display welcome dialog—enables/disables the Welcome dialog.
  • Default scripts directory—set the default location for opening/saving script files.
  • Default images directory—set the default location for exporting images.
  • Default videos directory—set the default location for exporting videos (not yet implemented).
  • Default stylesheet—set a stylesheet to use for all scripts. This is loaded after the factory stylesheet and before any script stylesheets.
  • Editor command—set the command to use to open a script file in an editor. If empty, the application will not attempt to open an editor. The sequence $F$ will be replaced by the file name.

Exit

Exits the program.

Window Menu

New Window

Creates a new window. The window will not have any file associated with it.

Help Menu

Sample Scripts...

Displays an Open File dialog in the installation's sample scripts directory. A sample script can then be selected the same as any other script file.

Contents

Displays the help content you are now viewing.

About Gamma

Displays the About dialog.
